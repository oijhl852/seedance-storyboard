# sd-segment-split — 15秒拆段

> 作者：Takis/安梓豪 | 仓库：[agent-skill-sync](https://github.com/oijhl852/agent-skill-sync)

根据戏剧节拍将剧本拆分为 4-15 秒的 Seedance 片段。

本 skill 是时长和镜头数量的唯一规则来源，其他子 skill 不自行修改片段边界。

## 拆段原则

- 15秒是上限不是目标；宁松勿紧
- 叙事段默认目标 10-15 秒；只有节拍完整或需要留白时才使用 4-9 秒
- 每段只承载一个主要戏剧任务
- 按节拍边界拆，不按时长硬拆

## 拆段判断

| 情况 | 处理 |
|------|------|
| 必须拆 | 长解释+对方回应、对话+肢体冲突+揭示、多人争吵、跨地点、闪回+现实动作 |
| 可同段 | 一句长台词+一个反应、2-4句短对白、对白+同情绪点OS、一个连续动作链 |

## 推荐段规模

| 段类型 | 推荐镜数 |
|--------|---------|
| 短对白（2-4句） | 4-5镜 |
| 长独白（预计超过8秒） | 3-4镜 |
| 快切蒙太奇 | 6-8镜，每镜1-2秒，总时长不超过15秒 |
| 情绪落版（OS） | 2-3镜 |
| 动作/打斗 | 5-6镜 |

## 反模式

| 错误 | 正确 |
|------|------|
| 长台词和短对白挤同段 | 长台词独占一段的镜1 |
| 快切蒙太奇拆分到两段 | 只要总时长不超过15秒就合并 |
| OS塞在段尾3秒 | OS独立成段 |
| 按时长强行拆段 | 按戏剧节拍拆 |

## 标准输出格式

每段必须输出：`片段编号`、`戏剧任务`、`预计时长`、`镜头数量`、`镜头列表`、`首帧状态`、`尾帧承接`、`拆分理由`。
