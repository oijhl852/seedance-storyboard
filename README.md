# Agent Skill Sync

> 作者：Takis

Seedance 2.0 视频分镜技能集合 — 从剧本到提示词再到面板注入的全流程自动化。

## 技能列表

| 技能 | 版本 | 说明 |
|------|------|------|
| **awesome-seedance** | v2.1.0 | 🧭 主编排层（入口） |
| sd-read-script | v1.0.0 | 📖 剧本读取与分场识别 |
| sd-segment-split | v1.0.0 | ✂️ 15秒拆段与戏剧节拍 |
| sd-dialogue | v1.0.0 | 💬 对话分镜铁律 |
| sd-action | v1.0.0 | ⚔️ 动作/打斗/蒙太奇分镜 |
| sd-story-adapt | v1.0.0 | 📝 故事→剧本改编 |
| sd-asset-guide | v1.0.0 | 🖼️ 资产构建指引 |
| sd-prompt | v1.0.0 | 🎯 提示词组装与16模板 |
| sd-prompt-library | v1.0.0 | 📚 精选案例库 |
| sd-community | v1.0.0 | 🌐 社区4776条提示词索引 |
| sd-quality | v1.0.0 | ✅ 质量检查 |
| sd-panel | v1.0.0 | 🔌 Chrome面板连接 |
| sd-chip | v1.0.0 | 🏷️ 资产Chip管理 |
| sd-inject | v1.0.0 | 💉 API注入引擎 |

## 知识来源

本技能集融合以下开源项目：
- [Seedance2-Storyboard-Generator](https://github.com/liangdabiao/Seedance2-Storyboard-Generator) — 分镜工作流
- [seedance-prompt-skill](https://github.com/songguoxs/seedance-prompt-skill) — 十大能力框架 (MIT)
- [awesome-seedance](https://github.com/ZeroLu/awesome-seedance) — 精选案例库 (CC BY 4.0)
- [awesome-seedance-2-prompts](https://github.com/YouMind-OpenLab/awesome-seedance-2-prompts) — 社区提示词 (CC BY 4.0)

## 备份

旧版技能见 `backup/` 目录：
- `backup/awesome-seedance-v2.0.1/` — 旧版单文件整合版
- `backup/seedance-browser-injector-v2.0.1/` — 旧版浏览器注入器

## 安装

```bash
git clone https://github.com/oijhl852/agent-skill-sync.git
```

然后使用 Reasonix 的 `install_source` 安装需要的 skill。
