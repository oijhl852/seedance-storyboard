# sd-action — 动作分镜

> 作者：Takis/安梓豪 | 仓库：[agent-skill-sync](https://github.com/oijhl852/agent-skill-sync)

专注动作戏、打斗、蒙太奇的分镜头设计。

## 动作线模板

```
先说明起始状态。随后描述发力和移动方向，再说明接触或变化，最后交代结果状态以及下一段承接状态。
```

## 写法要点

- 写连续动作链，避免切太碎
- 用完整句子串联连续动作，不使用箭头或符号代替动作变化
- 用可见的动作结果描述情绪：猛然转身、攥紧拳头、脚步停住；不要只写“愤怒地”“紧张地”
- 关键动作前重复当前身体状态（防止模型改姿势）
- 可润色动作连续性，不可改变剧情结果

## 蒙太奇

- 所有 B-roll 集中在一段
- 快切：6-8个镜头，每镜1-2秒，总时长不超过15秒
- 方便后期调用素材

## 打斗

- 5-6个镜头，每镜2-3秒；若动作链超过15秒，按动作结果拆段
- 关键物理动作给独立镜头

## 标准输出格式

每个动作片段输出：起始身体状态、连续动作句子、镜头列表、每个镜头的物理结果、受力/方向、尾帧承接状态。不能只输出形容词或情绪词，也不能用箭头串接动作。
