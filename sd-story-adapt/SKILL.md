# sd-story-adapt — 故事改编

> 作者：Takis/安梓豪 | 仓库：[agent-skill-sync](https://github.com/oijhl852/agent-skill-sync)

将小说、大纲或故事概念改编为分镜头剧本。仅当用户明确要求"帮我写剧本/改编故事"时使用。

## 输出结构

```
一、核心梗（2-4字）
二、故事梗概（背景/冲突/主角/主线/结局）
三、一句话卖点
四、人物小传（视觉形象/身份背景/核心标签/性格/金句）
五、剧本大纲（起承转合）
六、剧本正文（△分镜格式）
```

输出前先确认：题材、目标受众、单集时长、集数、结局倾向，以及用户是否允许改写原故事。
未确认的内容写入“待确认项”，不要为了套公式擅自改变人物关系或结局。

## 短剧公式

**单集结构**：冲突前置 + 情绪拉扯 + 阶段打脸 + 结尾留钩

**黄金三秒**：第一集开头通过强画面/强台词立刻抓住观众。

## 节奏控制

| 时长 | 结构 |
|------|------|
| 5集×15秒 | 简化三幕（起1→承/转2-3→合4-5） |
| 10集×15秒 | 标准三幕（起1-2→承3-6→转/合7-10） |
| 20集×15秒 | 四幕结构（起1-5→承6-10→转11-15→合16-20） |

## 交接给下游

改编完成后，必须把剧本正文交给 `sd-read-script` 重新读取，确认角色、场景、台词和动作都能被解析，再进入拆段。
